# ── InsightFace AntelopeV2 (REQUIS, à télécharger manuellement) ──────────────
#
#   Télécharger depuis :

#
#   Placer dans :
#   ~/.insightface/models/antelopev2/   (training)
#   ComfyUI/models/insightface/models/antelopev2/   (ComfyUI)
#
#   Fichiers requis (5 au total) :

#     - batch=8  : 24 Go  (RTX 3090, RTX 4090)
#     - batch=16 : 40 Go  (A100 — Vast.ai recommandé)
#
#   Phase 2 (IDFormer + PerceiverCA, Flux.2 gelé) :
